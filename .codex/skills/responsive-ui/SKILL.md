# Responsive UI Skill

Use this skill for MakaLearn interface work.

## Visual style

Use:
- Soft blue/white palette
- Teacher-professional layout
- Friendly learner activity areas
- Rounded cards
- Clear headings
- Large buttons
- Large tap targets
- High readability

Avoid:
- Tiny controls
- Desktop-only layouts
- Cluttered dashboards
- Generic dark SaaS styling
- Overly playful design that feels unprofessional

## Responsive requirements

Every page must work on:
- Desktop
- Tablet
- Mobile phones

Desktop:
- Sidebar navigation

Mobile:
- Bottom navigation or mobile menu
- Stacked content
- Large touch-friendly controls

Dashboards:
- Cards stack vertically on mobile.
- Charts remain readable.
- Tables should become cards or horizontally scroll.

Forms:
- Inputs stack on mobile.
- Upload controls remain usable on phone screens.
- Buttons should be easy to tap.

Gesture Practice:
- Desktop: split layout with camera, reference, and feedback visible.
- Mobile: stack camera, reference, controls, and feedback vertically.

Activities:
- Use large learner-friendly activity cards.
- Ensure drag/drop and quiz controls are usable on smaller screens.

## Accessibility settings

Include UI for:
- Large text mode
- High contrast mode
- Reduce motion
- Audio guidance toggle

These can use local state first.

## Feedback patterns

Use:
- Toast notifications
- Inline validation messages
- Loading states
- Helpful empty states

Use friendly, simple English.
