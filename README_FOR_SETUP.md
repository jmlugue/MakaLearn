# MakaLearn Codex Setup Files

This zip contains coding-agent instruction files for the MakaLearn project.

## Files included

```txt
AGENTS.md
.codex/skills/nextjs-app/SKILL.md
.codex/skills/makalearn-product/SKILL.md
.codex/skills/responsive-ui/SKILL.md
.codex/skills/future-supabase/SKILL.md
```

## Where to place these files

Extract the contents into the root of your Next.js project folder.

Example:

```txt
D:\MakaLearn\makalearn
```

After extracting, your project should look like:

```txt
D:\MakaLearn\makalearn
├─ AGENTS.md
├─ .codex/
│  └─ skills/
│     ├─ nextjs-app/
│     │  └─ SKILL.md
│     ├─ makalearn-product/
│     │  └─ SKILL.md
│     ├─ responsive-ui/
│     │  └─ SKILL.md
│     └─ future-supabase/
│        └─ SKILL.md
├─ package.json
├─ src/
└─ public/
```

## Important

If your project already has an `AGENTS.md`, open it first and decide whether to replace it or merge the content.

For the MakaLearn MVP, this included `AGENTS.md` is designed to be the main project instruction file.
