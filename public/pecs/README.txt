MakaLearn PECS/AAC Card Assets - Ready for Codex Upload

This ZIP contains only the finished generated card assets and the mapping files needed for the MakaLearn Playground feature.

Contents:
- generated_cards/ : final standardized PNG card images
- pecs_arasaac_manifest.csv : filename, label, category, and sentence_role mapping
- pecs_arasaac_manifest.json : same mapping in JSON format
- generation_results.json : source/processing record for the generated assets
- preview_contact_sheet.png : quick visual preview of the full set

Card style:
- white card
- black rounded border
- centered pictogram or simple text-only symbol where appropriate
- bold uppercase English word label
- no category tags, no role tags, and no badges inside the image

Notes:
- These are not official PECS cards.
- They are PECS-style/AAC cards formatted for the MakaLearn prototype.
